The canonical, per-version release notes for Label Alchemy live with the published extension, not
in these docs. This avoids a second copy that drifts out of date.

## Where to find release notes

- 📦 **VS Code Marketplace** — the [Changelog tab on the listing](https://marketplace.visualstudio.com/items/ramprasad94.labelalchemy/changelog)
- 🏷️ **GitHub Releases** — [github.com/Ramprasad94/labelalchemy/releases](https://github.com/Ramprasad94/labelalchemy/releases)
- 📣 **Announcements** — [Discussions → Announcements](https://github.com/Ramprasad94/labelalchemy/discussions/categories/announcements) for highlights of notable releases

If a GitHub Release is published for a version, it's the same content shown on the Marketplace.
